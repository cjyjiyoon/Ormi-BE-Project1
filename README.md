# 오르미 FE 프로젝트
## 1. 목표와 기능
### 1.1 목표
- HTML, CSS, JavaScript를 사용한 웹페이지 구현
- 호두페이지.....
### 1.2 기능
- 모바일 대응 페이지 제공
- 갤러리를 통한 사진 전시
- 이메일 유효성을 거친 이메일 전송 기능
- 스크롤 바에 따라 고정되는 내비게이션 바
- 페이지의 어디에서든 최상단으로 빠르게 이동할 수 있는 스크롤 탑 버튼
- 모바일 화면에서 슬라이드 메뉴 제공
